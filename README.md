# Smart-Security-Scanning-System
A smart security scanning system for real-time object detection and area monitoring.

# Project #3

# Project Overview
Smart Security Scanning System is a microcontroller-based surveillance project designed for real-time object detection and area monitoring. The system uses an ultrasonic sensor mounted on a servo motor to continuously scan its surroundings and detect nearby objects.

# Working Principle
When an object enters the predefined detection range (10 cm), the system immediately triggers visual and audio alerts through an LED and buzzer while displaying a warning message on a 16×2 I2C LCD screen. Once the object leaves the detection zone, the system automatically returns to normal scanning mode and resumes area surveillance.

# Key Features
• 180° Automatic Area Scanning
• Real-Time Object Detection
• Intruder Alert System
• LCD-Based Status Monitoring
• LED and Buzzer Notifications
• Automatic Return to Scanning Mode

# Components Used
• Arduino Nano
• HC-SR04 Ultrasonic Sensor
• SG90 Servo Motor
• 16×2 I2C LCD Display
• LED
• Buzzer
• Breadboard and Jumper Wires

# Applications
• Home Security Systems
• Restricted Area Monitoring
• Smart Surveillance Demonstrations
• Embedded Systems Learning Projects

# Conclusion
This project demonstrates the practical implementation of embedded systems by integrating sensing, scanning, alerting, and display technologies to create a compact and cost-effective security monitoring solution.
